## Variables

 - Variable names can only have letters, dollar signs, letters, numbers and underscore. 
 - Words are reserved by JavaScript and cannot be a variable name i.e, **const**, **let**, **function**, **if**, **else** etc.
				 *let x = 50
				 let y = 300
				 const z = 100
				 let name = "Harmon"*
	
You can write your variable in different styles:
										 - CamelCase : HoneyBee
										 - Snake_case : how_to_cook
										 - PascalCase : HowToCookRice
